League of Legends Performance Evaluator

This tool is used to view performance in recent League of Legends games.
Put your summoner name in the box and hit the button.
If you don't have a summoner name and still want to see this work, then just use one of these examples:

  • Natereater
  • 5E SoCal
  • From Iron
  • Doublelift
